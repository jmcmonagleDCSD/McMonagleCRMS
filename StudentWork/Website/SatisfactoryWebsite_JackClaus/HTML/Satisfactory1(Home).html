About

Satisfactory is a game available for Windows on Steam. The game is $29.99 usually. It is a factory building, sandbox type game. You are an employe of Ficit Corperation and were sent to this alien planet to extract its resources and make factories. You guid ADA will guide you through. Automate items to send them up through the Space Elevator for "Project Assembily" a mysteious project run by Ficit Corperation.
