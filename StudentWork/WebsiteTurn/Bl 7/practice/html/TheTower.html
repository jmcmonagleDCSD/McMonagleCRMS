Geometry Dash: The Tower




The Tower is a set of levels included only in Geometry Dash and Geometry Lite. The set includes 4 levels and once opened, it takes you to a tower where there are 4 doors reffering to the levels. If you opened the tower icon for the first time, only the first door will be available. Every door will open as you complete each level.
