The Game Modes

There are 5 game modes in rocket league.

Photo by deviantart.com from deviantart.com

Soccar

Soccar is the number 1 game mode, it has a ball that you try to score with. It features 1v1's, 2v2's, 3v3's, and 4v4's.

Snowday

Snow day is were there is a hockey puck and you try to score on the opponents goal.

Hoops

Hoops is a basketball game mode were you try andscore it above the net.

Rumble

Rumble is a powerup game mode. The powerups are, Magnitizer, The Boot, Haymaker, Tornado, Grappling Hook, Power Hitter, Disrupter, Plunger, Spike, Freezer, and Swapper.

Dropshot

Dropshot is a hexagon arena were there are tiles on the ground and once the ball hits one tile twice you can score there.
